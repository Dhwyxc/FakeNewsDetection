FAKE NEWS DETECTION EXTENSION

Fortunately, the browser still allows users to install the Extension from the outside very simply.
Step 1: Download the extension here, please proceed to extract the file to a folder.
Step 2: Access the Chrome Extension application manager by going to Menu -> Extensions -> Manage extensions
Step 3: Tick the "Developer Mode" and "Allow extensions from other stores" box.
Step 4: For zip extension installation you need to select "Load unpacked" and browse to the extracted folder of that Extension

After successfully installing the extension. Instructions for using the extension:
Step 1: When you want to check if a news is real or fake, highlight a piece of news.
Step 2: Click on the extension on the toolbar and select the model type to detect (recommended using RNN).
Step 3: Real or fake news prediction results will be displayed on the extension
